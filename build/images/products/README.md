# Product Images Folder

This folder contains product images for the store.

## Image Naming Convention

Images should be named according to the `image` field in `products.json`:

- `rice.jpg` or `rice.png` - for Basmati Rice
- `oil.jpg` or `oil.png` - for Sunflower Oil
- `sugar.jpg` or `sugar.png` - for Sugar
- `dal.jpg` or `dal.png` - for Toor Dal
- `tea.jpg` or `tea.png` - for Tea
- `biscuits.jpg` or `biscuits.png` - for Biscuits Pack
- `detergent.jpg` or `detergent.png` - for Detergent Powder
- `soap.jpg` or `soap.png` - for Soap Bar
- `cement.jpg` or `cement.png` - for Cement Bag
- `paint.jpg` or `paint.png` - for White Paint
- `hammer.jpg` or `hammer.png` - for Hammer
- `screwdriver.jpg` or `screwdriver.png` - for Screwdriver Set
- `wrench.jpg` or `wrench.png` - for Wrench Set
- `wire.jpg` or `wire.png` - for Electrical Wire
- `switch.jpg` or `switch.png` - for Switch
- `bulb.jpg` or `bulb.png` - for LED Bulb
- `pipe.jpg` or `pipe.png` - for PVC Pipe
- `tap.jpg` or `tap.png` - for Tap
- `fitting.jpg` or `fitting.png` - for Pipe Fitting Set
- `flour.jpg` or `flour.png` - for Wheat Flour

## Supported Formats

- `.jpg` / `.jpeg`
- `.png`
- `.webp`

## Image Guidelines

- Recommended size: 800x800px or 1000x1000px (square format)
- File size: Keep under 500KB for optimal performance
- Format: Use JPG for photos, PNG for images with transparency

## Adding New Images

1. Add your image file to this folder with the correct name
2. The image will automatically be used in the product cards
3. If an image is not found, a gradient placeholder will be shown
